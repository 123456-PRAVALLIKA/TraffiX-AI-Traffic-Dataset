# 🚦 TraffiX AI Traffic Optimization Dataset

This dataset is developed for the research project:

**"AI-Based Traffic Prediction and Optimization System (TRAFFIX)"**

---

## 📊 Dataset Overview
The dataset contains simulated traffic data representing multiple road intersections with AI-driven optimization parameters.

It is designed to support machine learning models for:
- Traffic prediction
- Signal timing optimization
- Reinforcement learning applications
- Smart city traffic management

---

## 🔢 Features Included
- Vehicle Count
- Average Speed (km/h)
- Traffic Density
- Signal Duration
- Pedestrian Count
- Weather Conditions
- AI Predicted Green Time
- Reinforcement Learning Reward
- CO2 Emissions (g)
- Fuel Consumption (Liters)
- Traffic Efficiency Score
